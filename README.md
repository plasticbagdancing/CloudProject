# Collaborative News Feed - Firebase Mini Challenge

## Team Members
- **Chanez Arroum**
- **Teodora Istrate**

## Description
A real-time collaborative news feed application built with Firebase Authentication and Firestore. Users can sign up, log in, and post messages that are instantly visible to all connected users.

## Features
- **Email/Password Authentication** (Sign up, Login, Logout)
- **Real-time Messaging** with live updates
- **User Identification** (author email + timestamp)
- **Responsive UI** with clean styling
- **Dynamic Interface** that updates based on auth state
